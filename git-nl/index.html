git-nl(1)

Aan de slag

  • git start

  • git voeg . toe
  • git voeg [-- <filepattern>] [--interactief] toe

  • git toestand

  • git neem op -m <msg>

  • git verschil <commit> <commit>

Vertakkingen

  • git vertak <new_branch>

  • git proef <branch>

  • git voeg <branch> samen

  • git verwijder vertakking <branch>

Samenwerken

  • git kloon <repository>

  • git haal binnen

  • git haal binnen en voeg samen

  • git stuur op

Duik er dieper in

  • git spaar

  • git pluk van <commit>

  • git opopop vergeet <pathspec>

  • git bizo start [<bad> [<good>]]

  • git keer terug naar <commit>

  • git stel in gebruiker.naam "Achterkat"
